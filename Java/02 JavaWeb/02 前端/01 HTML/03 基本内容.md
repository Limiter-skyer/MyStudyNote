# 属性
html元素可以设置属性，一般描述于开始标签。用name=“value”的方式。

```html
<a href="bing.com">超链接</a>
```

"href"就是"a"的属性。
一般包括在引号内。
属性对大小写不敏感，但是推荐用小写。

| 属性  | 描述                                     |
| ----- | ---------------------------------------- |
| class | 定义一个或多个类名（类名从样式文件导入） |
| id    | 定义元素的唯一id                         |
| style | 规定行内样式                             |
| title | 描述了额外信息                                         |

class能填多个，id只能填一个。
