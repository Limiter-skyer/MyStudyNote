# 步骤
1. 选择IDEA中File--->Settings
2. 搜索maven
3. 设置IDEA使用本地安装的Maven，并修改配置文件路径

![[Pasted image 20220725152729.png]]

# 坐标
资源的唯一标识，用来导入包
- groupId：当前Maven项目隶属组织名称（通常是域名反写)
- artifactId：当前项目名称（通常是模块名称）
- version：当前版本号
