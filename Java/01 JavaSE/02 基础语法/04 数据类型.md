# 基本数据类型
4大类8大种

| 数据类型 | 关键字       | 取值范围(byte) | 内存占用 |
| -------- | ------------ | -------------- | -------- |
| 整数     | byte         | -128~127       | 1        |
| 整数     | short        |                | 2        |
| 整数     | int(默认)    |                | 4        |
| 整数     | long         |                | 8        |
| 浮点数   | float        |                | 4        |
| 浮点数   | double(默认) |                | 8        |
| 字符     | char         |                | 2        |
| 布尔     | boolean      |                | 1        |

随便写一个整数，默认是int，如果想用定义一个Long整数，需要在后面加L/l。
同理小数与double亦如此，如果要定义float小数，要加F/f。
```java
long lg =3243562315234L;
float fl=999.99F;
```
