# 成员变量和局部变量

| 区别         | 成员变量             | 局部变量                         |
| ------------ | -------------------- | -------------------------------- |
| 类中位置不同 | 类中，方法外         | 方法中                           |
| 初始化值不同 | 有默认值，无需初始化 | 没有默认值，使用之前需要完成赋值 |
| 内存位置     | 堆内存               | 栈内存                           |
| 生命周期     | 随着对象存在         | 随着方法调用存在                 |
| 作用域       | 难以定义             | 在归属的大括号之中                                 |
